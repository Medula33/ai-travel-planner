# 🌍 AI Travel Planner

AI Travel Planner is a full-stack web application that helps users explore destinations and generate smart travel itineraries using AI. It provides a seamless experience for planning trips with personalized suggestions, destination insights, and an interactive dashboard.

## 🚀 Features
- Explore popular travel destinations with details and images  
- AI-powered itinerary generation for personalized trip planning  
- User authentication (Login/Signup system)  
- Interactive and responsive UI for smooth user experience  
- Organized dashboard to manage travel plans  

## 🛠️ Tech Stack
- Frontend: React.js, Vite, CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT  

## 📦 Setup Instructions
1. Clone the repository  
2. Install dependencies in both frontend and backend  
3. Run backend server using `npm start`  
4. Run frontend using `npm run dev`  

## 📌 Project Goal
To simplify travel planning by combining modern web technologies with AI to generate smart, personalized travel experiences.
